# Lưu trữ tài liệu legacy

Đây là nơi chứa tài liệu cũ chỉ để tra cứu khi cần đối chiếu lịch sử.

Quy tắc:

- Không dùng bất kỳ file nào trong `archive/` làm nguồn chuẩn.
- Nếu đã có file canonical mới thì ưu tiên file mới.
- Tài liệu ở đây có thể trùng lặp, lỗi thời, hoặc viết theo cấu trúc cũ.

## Nhánh legacy

- `archive/web-spec/` - spec cũ của dự án.
- `archive/architecture/` - ghi chú kiến trúc legacy.
- `archive/standards/` - quy ước boundary và authz legacy.
- `archive/notes/` - ghi chú phụ trợ và feedback legacy.
- `archive/system-architecture.md` - summary platform legacy.
