# Lưu trữ tài liệu legacy

Chứa tài liệu cũ, chỉ tra cứu khi cần đối chiếu lịch sử.

Quy tắc:

- Không dùng file trong `archive/` làm nguồn chuẩn.
- Nếu có file canonical mới, ưu tiên file mới.
- Tài liệu có thể trùng lặp, lỗi thời, hoặc cấu trúc cũ.

## Nhánh legacy

- `archive/web-spec/` - spec cũ
- `archive/architecture/` - ghi chú kiến trúc legacy
- `archive/standards/` - quy ước boundary + authz legacy
- `archive/notes/` - ghi chú phụ trợ + feedback legacy
- `archive/system-architecture.md` - summary platform legacy
