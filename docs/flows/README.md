# Tài liệu flow

Thư mục này giữ user flow và operational flow canonical.

Quy tắc:

1. Flow mô tả thứ tự hành vi và state transition ở mức sản phẩm.
2. Flow không lặp lại toàn bộ functional requirement.
3. Flow phải trỏ về PRD và requirement liên quan.
